## Shared Preferences

Данные в виде пар `key`-`value`.  


Находятся в папке приложения:  
`/data/data/YOUR_PACKAGE_NAME/shared_prefs/YOUR_PREFS_NAME.xml`

<br>

*Use case*:  
Настройки пользователя и маленькие наборы данных.  

<br>

|   key    |  value       |
| -------- | --------     |
|`String`  |  `int`       |
|          |  `long`      |
|          |  `float`     |
|          |  `String`    |
|          |`Set<String>` |
