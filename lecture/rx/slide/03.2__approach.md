# Стандартный подход

* Делаем запрос и ждем **данные**
* Источники данных не связаны между собой
* Храним состояние и сверяемся с ним

# Реактивный подход

* Делаем запрос и ждем **результат**
* Источники данных связаны
* Получаем изменения состояниям
